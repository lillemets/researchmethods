Section 11 Model transformations

11.1 Multiple model

More than one predictor.

\(Y = \alpha + \beta_{1} x_1 + \beta_{2} x_2 + \varepsilon\)

Why use multiple predictors?

See Navarro and Foxcroft (2018) section 12.5 for an introduction and Crawley (2013) section

11.2 Categorical predictors

Interpretation:

The first factor level (High) is the base. All other factor values are compared against this base value.

When weight1 is 0 and diet is High, weight2 is 171.48.

When weight1 increases by 1 unit, weight2 increases by 1.226 units.

When diet is Medium, weight2 is lower by 36.368 units compared to when diet is High.

When diet is High, weight2 is no different compared to when diet is High. This is because the coefficient is not statistically significant(ly different from 0).

11.3 Transformations

To improve fit of linear model.

11.3.1 Log-transformations

When the distribution is highly skewed, we may wish to use (natural) logged values of variable(s).

11.3.1.1 Log-linear

\(ln(Y) = \alpha + \beta_{1} x + \varepsilon\)

Interpretation:

\[ln(weight2_{i}) = 6.105 + 0.001 * weight1_{i} + \varepsilon_{i}\]

When weight1 increases by 1 unit, weight2 increases by 0.001165 * 100 = 0.117 percent.

11.3.1.2 Linear-log

\(ln(Y) = \alpha + \beta_{1} ln(x) + \varepsilon\)

Interpretation:

\[weight2_{i} = -4298.479 + 812.191 * ln(weight1_{i}) + \varepsilon_{i}\]

When weight1 increases by 1 per cent, weight2 increases by 812.191 / 100 = 8.122 units.

11.3.1.3 Log-log

\(ln(Y) = \alpha + \beta_{1} ln(x) + \varepsilon\)

Interpretation:

\[ln(weight2_{i}) = 1.639 + 0.807 * ln(weight1_{i}) + \varepsilon_{i}\]

When weight1 increases by 1 per cent, weight2 increases by 0.807 percent.

11.3.2 Polynomials

See Crawley (2013) section 10.3.

\(Y = \alpha + \beta_{1} x + \beta_{2} x^2 + \varepsilon\)

Interpretation:

\[weight2_{i} = -106.134 + 2.047 * weight1_{i} + -0.001 * weight1_{i}^{2} + \varepsilon_{i}\]

None of the coefficients are statistically significant(ly different from 0). Since we weight1 has more than one coefficient, it would be difficult to interpret anyway.
